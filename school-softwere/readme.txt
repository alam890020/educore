A powerful WordPress plugin to manage multiple schools with students, staff, exams, fees, attendance, library, transport, hostel and more in one beautiful dashboard.

== Description ==

School Softwere is a comprehensive school management system for WordPress. It provides:

* Multi-school management
* Student admissions, profiles, ID cards
* Staff management with roles & permissions
* Classes, sections, subjects, timetables
* Examinations, results, admit cards
* Fees, invoices, payments, concessions
* Attendance (student & staff)
* Library, transport, hostel
* Notices, events, homework
* Reports, certificates
* Student & parent portal (frontend)

== Installation ==

1. Upload the `school-softwere` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Run the Setup Wizard from the School Softwere menu.
